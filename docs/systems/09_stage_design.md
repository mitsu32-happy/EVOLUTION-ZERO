# Stage Design v0.1

## Core Philosophy
ステージは探索空間ではなく、進化戦場。

## Map Structure
固定地形 + ランダムイベント + ランダム敵構成。

## Recommended Map Feel
広い一本道に近い開けた戦場。  
複雑地形はストレスになりやすいため避ける。

## Map Type
無限ループ型を推奨。  
ブラウザ・スマホ・GitHub Pagesと相性が良い。

## Stage Flow
1. 序盤: 少数の弱敵
2. 中盤: 群れ増加
3. エリート出現
4. イベント発生
5. 進化タイミング
6. 大群
7. ボス

## Stage Themes
### Jungle
基本ステージ。

### Volcano
爆発、高火力、狂暴適応。

### Swamp
毒敵、移動低下、毒進化が出やすい。

### Meteor Field
特殊DNA、危険変異、DNA暴走。

### Frozen Area
低視界、鈍化、耐久系。

### Ancient Ruins
高難易度、古代適応。

## Events
- 隕石イベント
- DNA暴走区域
- 古代遺骸
- 群れ暴走

## Key Principle
背景だけ変えるのではなく、ステージごとにプレイ感と進化傾向を変える。
