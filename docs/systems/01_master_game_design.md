# EVOLUTION ZERO Master Game Design v0.1

## Genre
恐竜進化サバイバルアクション（ヴァンサバライク）

## Core Theme
絶滅後の原始世界で、恐竜がDNA適応と突然変異を繰り返し、最強の捕食者へ到達する。

## Game Identity
EVOLUTION ZEROは「恐竜版ヴァンサバ」ではなく、「進化を発見するサバイバルアクション」として設計する。

## Core Promise
- 大量殲滅の爽快感
- 進化直前の期待感
- 未知進化を発見する驚き
- 恐竜ごとに異なるプレイ感
- DNA研究による長期的な周回動機

## Player Emotion Curve
### Early Game
弱い状態で逃げながら戦い、ギリギリ生き残る。

### Mid Game
スキルが揃い始め、進化条件を意識してビルドを組む。

### Late Game
進化・覚醒し、大群を蹂躙しながらボスに挑む。

## World Direction
- 絶滅後の原始世界
- 隕石後の異常進化
- DNA暴走
- 極限生態系
- 捕食競争

## Design Tone
リアル寄り恐竜 + 少しSF。  
恐竜の魅力を消さないため、完全サイバーSFには寄せすぎない。

## Core Differentiator
スキル構成が進化分岐を誘発する。  
新スキル解放によって新たな進化ルートが発見可能になる。

## MVP Goal
コンテンツ量ではなく、「進化が気持ちいい」「もう1回遊びたい」を最初に成立させる。
