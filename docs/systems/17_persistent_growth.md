# Persistent Growth System v0.1

## Core Philosophy
永続成長は強くなるためだけではなく、新しい進化を発見するための研究。

## Main Currency
DNAポイント。

## Uses
- 新スキル解放
- 恐竜解放
- 基礎能力強化
- パッシブ適応強化

## Priority
1. 新スキル解放
2. 恐竜解放
3. パッシブ適応
4. 基礎能力強化

## New Skill Unlock
新スキルが増えることで、関連する進化ルートが自然に増える。

## Dinosaur Unlock
新しい戦い方の追加として扱う。

## Base Upgrades
控えめにする。
- 最大HP
- 攻撃力
- 移動速度
- EXP取得範囲
- DNA獲得量

## Passive Adaptation
- 初期スキル候補+1
- レベルアップ時のリロール回数+1
- レアスキル出現率UP
- 進化ヒント表示精度UP
- ボスDNA獲得量UP

## Key Principle
数字を伸ばすゲームではなく、発見とビルドの幅を広げるゲームにする。
