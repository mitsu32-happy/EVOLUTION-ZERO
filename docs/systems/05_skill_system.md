# Skill System v0.1

## Core Philosophy
スキルは攻撃手段であり、進化素材でもある。  
EVOLUTION ZEROでは、スキルは「進化DNA」として扱う。

## Slots
- 通常スキル: 3枠
- 必殺技: 1枠
- パッシブ適応: 別管理

## Why 3 Skill Slots
- スマホで管理しやすい
- ビルドの方向性が見えやすい
- 進化条件が分かりやすい
- UIが崩れにくい

## Ultimate Skill
必殺技は手動発動。  
移動メインのゲームに「ここで使う」という判断を加える。

## Skill Categories
### OFFENSE
牙撃、咆哮、衝撃波、引き裂き

### SPEED
瞬足、加速、回避本能、残像

### DEFENSE
硬化皮膚、棘装甲、再生、挑発

### STATUS
毒液、感染、流血、麻痺

### INSTINCT
狩猟本能、狂暴化、群れ適応、捕食進化

## Synergy Examples
- 流血 + 多段ヒット = 流血爆発
- 毒 + 感染拡散 = 全体腐敗
- 高速 + 回避 = 分身発生

## Skill Level
推奨 Lv1〜5。  
Lv5では数値だけでなく、範囲追加・属性追加・特殊効果追加など挙動を変える。

## Level-Up Choice
3択。  
各カードには、アイコン、スキル名、効果、DNA適応タグ、シナジーを表示する。

## Reroll
選択自体はスキップ不可。必ずどれかを選ぶ。  
リロールは「救済」ではなく「戦略資源」として扱う。

## Skip
選択スキップは採用しない。  
ただしレベルアップ演出スキップは採用可能。
