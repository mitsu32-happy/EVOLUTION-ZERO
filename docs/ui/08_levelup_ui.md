# Level-Up UI v0.1

## Core Role
レベルアップは単なる強化ではなく、DNA適応選択。

## Player Emotion
「どの進化に向かう？」と悩ませる。

## Basic Layout
- 3択カード固定
- 背景は暗転
- 背後のプレイ画面は停止
- スマホ縦で見やすく押しやすい配置

## Card Content
- アイコン
- スキル名
- 効果
- DNA適応タグ
- シナジー表示
- レア度表現

## Evolution Hints
答えを全部言わない。  
例: 「高速遺伝子が強く反応している…」

## Presentation Timing
演出はテンポを壊さない。

推奨:
1. 時間停止: 0.05〜0.1秒
2. 軽い暗転: 0.1秒
3. DNAライン展開: 0.2〜0.3秒
4. カード出現

合計0.5秒前後。

## Skip Policy
選択スキップは不可。  
演出スキップのみ可能。タップまたは長押しで即カード表示。

## Reroll
選ばされるからこそ価値がある。  
リロールは「救済」ではなく「戦略資源」。

## Rare Choices
レアDNAや危険変異はノイズ、赤警告、DNA暴走で特別感を出す。
