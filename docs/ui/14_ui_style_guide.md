# UI Style Guide v0.1

## Core Direction
未来SFではなく、危険生物研究UI。

## Keywords
原始、DNA、危険進化、生体反応、琥珀、骨格、発光血管。

## Main Colors
- Deep Green: 原始感、密林感、生体感
- Amber Gold: DNA、化石、古代感
- Warning Red: 進化暴走、危険、必殺、ボス
- DNA Blue: 解析、研究、レア

## Texture
半透明、生体発光、骨格モチーフ、DNAライン。

## Frames
六角形、骨、DNAラインをベースにする。

## Buttons
「押す」ではなく「起動する」感。  
タップ時は発光、DNA脈動、微振動、琥珀発光。

## Fonts
- 英語: 見出し、進化名
- 日本語: 説明、システム
- スマホで読みやすい太めのフォント

## Animation
UIが生きている感を出す。  
微脈動、DNA流動、発光、呼吸、警告ノイズ。  
ただし常時ギラギラは避ける。

## Effects
通常時は控えめ。  
進化時・必殺時・ボス時のみ暴走させる。

## Screen Policy
基本は1画面完結。  
スクロールなしで即判断できるUIを優先する。
