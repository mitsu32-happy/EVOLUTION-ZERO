# お供恐竜UI仕様 P01

## ホーム

- ホーム中央の称号付近に小型の「お供」パネルを表示する。
- セット中は `表示名 Lv` を表示する。
- 卵所持中は `卵あり`、孵化中は `孵化中`、未所持/未セットは `お供なし` と表示する。
- パネルをタップ/クリックすると、お供恐竜選択モーダルを開く。

## お供恐竜選択モーダル

P01は最小構成とする。

- 所持済みお供恐竜一覧
- セット中表示
- 効果概要
- DNAによる強化ボタン
- 範囲外タップで閉じる

個別アセットはP02以降で差し替えるため、P01では代表アイコンを使用する。

## 研究画面

身体強化カテゴリに、卵所持/孵化中のみ孵化カードを表示する。

- `卵を孵化`: DNA/研究Ptを消費して孵化開始
- `卵を孵化中`: 完了までの残り時間を表示
- `卵の孵化完了`: 受け取り操作で未所持からランダム取得

## リザルト

プレイ中に卵を取得した場合、報酬行に `卵入手 / お供恐竜の卵` を表示する。

## プレイ中

セット中のお供恐竜をプレイヤー周辺に小さく表示する。プレイヤーと完全に重なり続けないよう、ゆるい周回追従にする。

## チュートリアル/案内

- 卵取得時: `卵を入手しました。研究画面で孵化できます。`
- 研究画面: 孵化カードの説明で孵化コストと効果を表示
- ホーム: 既存ホームチュートリアルにお供恐竜ステップを追加
- セット後: ホーム通知でセット完了を表示
