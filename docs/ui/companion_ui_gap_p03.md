# MVP-P03 お供恐竜 UI/UX gap

## ホームお供パネル

現状:
- ホーム中央付近に、お供名、Lv、卵あり/孵化中/お供なしを表示する。
- セット中のiconを表示できる。

課題:
- 既存ホームUIに情報が増え、スマホ幅で密度が高くなりやすい。
- 本番アセットに合わせたpanel/selected表示が必要。
- チュートリアルのハイライト対象として登録済みだが、本番位置の最終確認が必要。

## お供選択/強化モーダル

現状:
- ホームのお供パネルから開く。
- 所持済みお供を行として表示し、タップでセット、ボタンで強化できる。

課題:
- 行が5つ固定で、10種類を所持した場合に全件を表示できない。
- スクロールまたはページングが本番前必須。
- 行の説明文が長い場合にスマホで読みにくい。
- コントローラー/仮想マウス操作の検証が必要。
- 強化後の効果差分が見えづらい。

## 卵所持/孵化中/孵化完了表示

現状:
- ホームパネルと研究画面孵化カードで状態を示す。
- 研究画面でDNA/研究Ptを消費し、孵化を開始/完了できる。

課題:
- 卵取得後にどこへ行けばよいか、リザルトから研究への誘導が弱い。
- 孵化完了時の演出が通知中心で、達成感が弱い。
- 孵化中の残り時間表示をPWA復帰時に再確認する必要がある。

## リザルト卵表示

現状:
- 卵取得時に報酬行として表示する。

課題:
- 卵が何に使えるかの説明が少ない。
- 取得演出と研究誘導を強化したい。

## チュートリアル

現状:
- 卵取得、研究画面での卵孵化、お供取得後、セット後の通知文言がある。

課題:
- A13系ハイライトチュートリアルに統合できていない。
- 再表示/リセット導線との整合確認が必要。
- 初回卵取得から孵化までが時間経過を挟むため、説明を分割する必要がある。

## 本番前必須UI

1. お供一覧のスクロール/ページング。
2. 選択中/未所持/強化可能/MAXの状態表示。
3. 卵孵化カードの専用パネル化。
4. 孵化完了演出と取得結果表示。
5. スマホ幅でのテキスト折り返し確認。
6. コントローラー/仮想マウスで選択/強化/閉じる操作ができること。
