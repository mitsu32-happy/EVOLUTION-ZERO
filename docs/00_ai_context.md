# EVOLUTION ZERO AI Context v0.1

## Project
EVOLUTION ZERO

## Core Concept
恐竜進化サバイバルアクション。ヴァンサバライクをベースに、恐竜・進化・DNA研究・発見型ビルドを中核にする。

## Core Experience
大量の敵を蹂躙しながら、スキル選択によってDNA適応を進め、未知の進化を発見し、最強の捕食者へ到達する。

## Target Platform
- Browser game
- HTML / CSS / JavaScript
- GitHub Pages publishing
- Mobile-first, portrait screen
- 1画面完結UIを基本とし、重要画面ではスクロールを極力使わない

## Visual Direction
- Top-down 2.5D
- 高品質2D素材で2.5Dに見せる
- 深緑、琥珀ゴールド、赤警告、DNAブルー
- 未来SFではなく「危険生物研究UI」
- 恐竜はリアル寄り、UIと演出に少しSFを混ぜる

## Controls
- 攻撃は基本オート
- 移動は画面内の非HUD領域ならどこでもドラッグ操作可能
- スティック表示位置は固定でもよいが、「操作領域は画面内どこでも可能」と仕様上明確にする
- 必殺技のみ手動発動

## Important Product Principles
- 進化は直接アンロックしない
- 新スキル解放によって進化ルートが増える
- 進化条件はすべて公開せず、発見の楽しさを残す
- 進化は単なるステータス強化ではなく、見た目・戦い方・スキル挙動が変わるものにする
- Codexには仕様・画像参照・対象範囲を明確にした小タスクで実装させる
