# Evolution Sequence v0.1

## Core Role
進化演出はゲーム最大の報酬。  
強化ではなく覚醒として見せる。

## Recommended Duration
2〜4秒。  
短く、派手に、気持ちよく。

## Sequence
1. 異常反応
   - 画面ノイズ
   - DNA暴走
   - 心音
   - HUD警告

2. 時間停止
   - 0.2〜0.4秒
   - 特別感を出す

3. DNA展開
   - DNA螺旋
   - 骨格投影
   - 発光血管
   - 進化シルエット

4. 変異
   - 骨格変形
   - 棘展開
   - 爪変形
   - 肉体肥大
   - 生物感を重視

5. 咆哮解放
   - 全画面振動
   - 敵吹き飛び
   - 衝撃波
   - 周囲殲滅

6. 新HUD移行
   - HPバー変化
   - 必殺変化
   - エフェクト変化

## Important Notes
- 完全暗転しない。プレイ中に起きている感を残す。
- 進化中は世界を停止し、演出中死亡を防ぐ。
- 進化後数秒は無敵または大幅強化で報酬感を出す。
