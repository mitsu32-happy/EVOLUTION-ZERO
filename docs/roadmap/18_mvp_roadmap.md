# MVP Roadmap v0.1

## MVP Goal
「進化が気持ちいい」「もう1回遊びたい」を成立させる。

## MVP Required
1. プレイ画面
2. 敵スポーン
3. レベルアップ
4. スキル選択
5. 進化
6. ボス
7. DNA研究

## MVP Defer
- 多数恐竜
- 多数進化
- 多数ステージ
- 図鑑
- 実績
- 複雑イベント
- オンライン

## MVP Content
### Dinosaurs
3体:
- Raptor
- T-Rex
- Ankylosaurus

### Evolutions
各恐竜2〜3分岐。

### Skills
15〜20個。

### Stages
1〜2個:
- Jungle
- Volcano

### Bosses
2〜3体。

## Development Phases
### Phase 1: Play Foundation
移動、敵、攻撃、EXP。

### Phase 2: Level-Up
3択、スキル、リロール。

### Phase 3: Evolution
進化条件、進化演出、進化後挙動。

### Phase 4: Boss
ボス出現、攻撃、報酬。

### Phase 5: DNA Research
永続DNA、新スキル解放、恐竜解放。

### Phase 6: Presentation
HUD、演出、音、UI polish。

## MVP Success Conditions
- 進化が気持ちいい
- 別ビルドを試したい
- 別恐竜を触りたい
- DNA研究を進めたい
