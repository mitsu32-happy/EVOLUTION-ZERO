- near-magenta pixels: likely #ff00ff chroma remnants.
- cyan fringe pixels: possible generated/cutout edge glow that may need manual review.

## Notes

- Boss/effect sheets can intentionally include glow; review visually before regenerating.
- High-priority sheet cell-edge contact is most important for animation frames because it can create visible clipping or neighbor-frame bleed.
