# EVOLUTION ZERO Balance Snapshot

MVP-038 does not rebalance the game. It documents the current tuning points so future balance work can be deliberate.

## Core Rules

- Play-run growth is temporary.
- Permanent growth belongs to DNA research.
- Evolution branches by adaptation tags. It must not read as a single linear evolution path.
- Dinosaur selection is a role choice, not a permanent character level.

## Current Role Map

| Area | Role | Notes |
| --- | --- | --- |
| Velociraptor | Speed | Fast movement and attack tempo |
| Triceratops | Defense | High HP and damage reduction |
| Tyrannosaurus | Attack | Higher damage and range |
| Swarm enemy | Crowd pressure | Hunting evolution should feel good |
| Fast enemy | Reaction pressure | Speed evolution should answer it |
| Tank enemy | Durability wall | Attack evolution should answer it |

## Future Balance Checks

- Does HARD feel noticeably dangerous without becoming unfair?
- Does fast enemy pressure feel avoidable with the wider camera?
- Does tank reward justify time spent attacking it?
- Does DNA research make repeat runs feel better without implying character levels?
- Do evolution branches create different preferred situations?
